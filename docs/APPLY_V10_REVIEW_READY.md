# v10 적용 순서 (Review Ready)

## 1) 프론트: 법적 페이지/푸터 링크 확인
- `/terms`, `/privacy`, `/support`, `/help` 는 **로그인 없이 접근 가능**
- 모든 주요 화면 하단에 `LegalFooter` 가 표시되어야 함

## 2) 문구 가이드 (반려 방지)
- “보증금/상금/배팅/분배” 같은 단어는 MVP 단계에서 제거
- “참가비/리워드/환급(리워드)” 중심으로 표기
- 돈을 벌게 해준다는 표현 금지 (습관 관리/리워드 성격 강조)

## 3) 앱인토스 테스트/검토 요청
- 콘솔 업로드 후 QR 테스트 **최소 1회 이상** 필수
- 테스트는 **워크스페이스 멤버 + 만 19세 이상** 조건 존재

## 4) API 연동 확대 시
- 앱인토스 API(토스 로그인/포인트 지급 등) 호출하려면 **mTLS 인증서**를 서버에 적용해야 함
